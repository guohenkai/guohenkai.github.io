2、分析 Ant Design Pro 自带的 models

位置：”/src/models/“

分别为：global.js 、list.js 、login.js 、menu.js 、project.js 、setting.js 、user.js

熟悉它们，对于我们以后调整或者修改基本配置，很有帮助。

Ant Design Pro 有自己一套的加载数据体制，我们可以在预编译文件可以看到。
位置：”/src/pages/.umi/initDva.js”

细心的小伙伴，也发现了有 dispatch 函数，它是什么呢？如何使用它呢？

4、dispatch 使用

dispatch 是一个用于触发 action 的函数，action 是改变 State 的唯一途径，但是它只描述了一个行为。而dispatch可以看作这个行为的的方式。

在 dva 中，connect Model 的组件，通过 props 可以访问到 dispatch ,可以调用 Model 中 Reducers 和 Effects

在Model user 的 add 方法代码下方添加：if(callback) callback();
