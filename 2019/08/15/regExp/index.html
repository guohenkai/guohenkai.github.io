正则表达式总结

发表于 2019-08-15   |   分类于 javaScript   |  

正则表达式，又称规则表达式。（英语：Regular Expression，在代码中常简写为regex、regexp或RE），计算机科学的一个概念。正则表达式通常被用来检索、替换那些符合某个模式(规则)的文本。

  • 用在前端，就一句话：匹配字符串。

正则表达式中的特殊字符

  • 边界符

    • ^ 匹配开始
    • \$ 匹配结尾

  • 字符类

    • [] 匹配其中一个
    • 注意：方括号内部加上^表示取反，只要包含方括号内的字符，都返回 false

  • 量词符

    • * 重复0次或更多次 【>=0 次】
    • + 重复1次或更多次 【>=1 次】
    • ? 重复0次或1次
    • {n} 重复n次
    • {n,} 重复n次到更多次
    • {n,m} 重复n次到m次
    • 注意：方括号内部加上^表示取反，只要包含方括号内的字符，都返回 false

  • 预定义类

    • \d \D 数字
    • \w \W 数字字符下划线
    • \s \S 换行符，空格，制表符

修饰符

  • g 全局替换
  • i 忽略大小写
  • gi
